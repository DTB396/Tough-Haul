# ========================================
# TOUGH HAUL - SITE OWNER DATA FILES
# ========================================
# 
# This folder contains all the data files you can edit to update
# your website content WITHOUT touching any code files.
#
# ========================================
# WHAT'S IN THIS FOLDER
# ========================================
#
# 1. services.yml     - Your service offerings (junk removal, furniture pickup, etc.)
# 2. reviews.yml      - Customer testimonials and ratings
# 3. faq.yml          - Frequently Asked Questions
# 4. portfolio.yml    - Before/after photos, completed jobs
#
# ========================================
# HOW TO EDIT
# ========================================
#
# 1. Open any .yml file in a text editor (Notepad, VS Code, etc.)
# 2. Follow the existing format carefully
# 3. Change text between quotes "like this"
# 4. Save the file
# 5. Run: npm run build
# 6. Run: git add -A
# 7. Run: git commit -m "Updated [what you changed]"
# 8. Run: git push origin main
#
# ========================================
# IMPORTANT RULES
# ========================================
#
# - Keep the same YAML structure (spaces, dashes, colons)
# - Don't change the field names (id:, title:, description:)
# - Always use quotes around text with special characters
# - Use 2 spaces for indentation (not tabs)
# - Test locally before pushing: npm run dev
#
# ========================================
# QUICK EDITS
# ========================================
#
# ADD A SERVICE:
# - Open services.yml
# - Copy an existing service block
# - Paste it at the bottom
# - Change the id, title, description, price
#
# ADD A REVIEW:
# - Open reviews.yml
# - Copy a review block
# - Paste and edit name, rating (1-5), text, date
#
# ADD A FAQ:
# - Open faq.yml
# - Copy a question block
# - Paste and edit question + answer
